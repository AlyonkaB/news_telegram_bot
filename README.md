# news_telegram_bot

## Installation

```shell
git clone https://github.com/AlyonkaB/news_telegram_bot.git
cd news_telegram_bot/
python -m venv venv
source venv/scripts/activate
pip install -r requirements.txt
python news_tele_bot.py
```


## Create a new file named `.env`

# Telegram_bot
